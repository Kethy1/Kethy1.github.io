## 🎯 项目简介 | Project Overview

这是一个**国家级大学生创新训练项目**，旨在为大学生提供个性化的求职咨询和职业规划建议。通过对LLaMA模型进行微调，系统能够理解复杂的学习与求职场景需求，提供高质量的决策支持。

A national-level innovation project that leverages fine-tuned LLaMA to provide college students with personalized career consulting and job hunting advice. The system integrates robust database design, backend services, and intelligent recommendation capabilities.

## 🛠️ 技术栈 | Tech Stack

- **LLM Framework**: LLaMA (Fine-tuned with custom career Q&A dataset)
- **Backend**: Python Flask/FastAPI
- **Database**: Custom-built SQL database with career, recruitment, and industry data
- **Frontend**: WeChat mini-program for user consultation
- **Deployment**: Cloud-based or local server deployment

## 📊 核心功能 | Key Features

1. **岗位匹配** - Job-position matching based on skills and interests
2. **职业规划** - Career path recommendations and development planning
3. **行业洞察** - Industry trends and salary information
4. **个性化建议** - Personalized guidance tailored to individual profiles
5. **学习资源** - Curated learning resources and skill development paths
